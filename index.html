What is Pomofocus?

Pomofocus is a customisable pomodoro timer that works on desktop &mobile browser. The aim of the app is to help you focus on any taks you are working on, such as study, writing, or coding. This app is inspired by Pomodoro Tequinique which is a time management method development by Francesco Cirillo.

What is Pomodoro Tequinique?

The Pomodoro is created by Francesco Cirillo for a more productive way to work and study. The tequinique uses a timer to break down work into intervals, traditionally 25 minutes in lenght, seperated by short breaks. Each interval is known as a pomodoro, from the Italian word for tomato, after the tomato-shaped kitchen timer that cirillo used as a university student. - Wikipedia

How to use the Pomodoro Timer?
  1. Add task to work on today
  2. Set estimate pomodoros (1 = 25min of work) for each tasks
  3. Select a task to work on
  4. Start timer and focus on the task for 25 minutes
  5. Take a nreak for 5 minutes when the alarm ring
  6. Take a break for 5 minutes when the alarm ring
  7. Iterate 3-5 until you tinish the tasks
Features
  • Responsive design that works with the desktop and mobile
  • Color transsition to swich moods between work time and rest time
  • Audio notification at the end of timer period
  • Customizable timer intervals to suit your preference
